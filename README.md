# Drone Mini-Game

A small controllable drone prototype.

## Controls

- **E** — Grab / Drop  
- **W A S D** — Move  
- **Space / C** — Move Up / Down  
- **LMB / RMB** — Rotate  
- **V** — Switch Camera  
- **Mouse Button 4 / 5** — Camera Tilt Up / Down
